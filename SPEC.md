# 生日祝福网站 - 技术规范

## 1. 项目概述

**项目名称**: Birthday Wishes - 精美生日祝福网站

**核心功能**: 输入姓名和生日 → 进入祝福页面 → 蛋糕逐渐完成 + 生日贺卡 + 庆祝动画

**目标用户**: 送给朋友/亲人的生日惊喜

### 视觉设计

**色彩方案**:
- 主背景: `#0f0c29` → `#302b63` → `#24243e` (紫黑渐变)
- 主强调: `#ff6b6b` (珊瑚红)
- 次强调: `#feca57` (金黄)
- 卡片背景: `rgba(255,255,255,0.1)` (毛玻璃)
- 文字主色: `#ffffff`
- 文本次色: `rgba(255,255,255,0.8)`

**字体**:
- 标题: "Dancing Script"/"Playfair Display" (手写风格)
- 正文: "Quicksand"/"Nunito" (圆润现代)
- 数字: "Poppins" (几何感)

**动画效果**:
- 星空粒子背景漂浮
- 蛋糕逐层点亮动画
- 烟花/彩纸庆祝特效
- 卡片3D翻转效果
- 按钮悬停发光

## 3. 功能规范

### 模块1: 输入页面
- 输入框: 姓名 (必填)、生日日期 (YYYY-MM-DD)
- 背景: 星空粒子动画 (纯CSS/Canvas)
- 按钮: 带有光晕效果的"开始祝福"
- 验证: 姓名不能为空

### 模块2: 蛋糕动画
- 3层蛋糕图案 (使用CSS绘制或SVG)
- 每层带有蜡烛
- 点击蜡烛后逐层点燃
- 全部点燃后播放庆祝特效
- 蛋糕完成时的彩带烟花效果

### 模块3: 生日贺卡
- 3-5张祝福卡片
- 每张包含:
  - 祝福语
  - 小装饰 (星星/气球/蛋糕/爱心)
- 支持点击翻页 / 滑动翻页
- 卡片有3D翻转效果

### 模块4: 音乐控制
- 背景音乐: "生日快乐" 歌
- 音频控制按钮 (播放/暂停)
- 可选: 语音祝福 (可选功能)

### 模块5: 庆祝特效
- 彩纸/彩带从天而降
- 烟花动画
- 星星闪烁效果

### 有趣附加模块
- **心愿瓶**: 点击写下心愿，瓶子收集心愿
- **时光胶囊**: 展示还可以活多少天(如果输入的是未来生日)
- **照片墙**: 可以上传展示合照 (本地预览)
- **弹幕祝福**: 观众可以发送弹幕祝福 (可选实时功能)

## 4. 技术实现

### 技术栈
- 纯HTML + CSS + JavaScript (无框架)
- Canvas绘制粒子背景
- CSS动画为主
- 单页面应用 (SPA)
